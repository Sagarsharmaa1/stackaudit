# StackAudit Complex Logic Documentation

## Score Calculation
- Starts at 100
- Deduct 5 for each major update
- Deduct 2 for each minor update
- Deduct 0.5 for each patch
- Deduct 5 for bundle size > 500kb gzip
- Deduct 10 if total dependencies > 50
- Deduct 15 if total dependencies > 100

## Report Encoding
The report data is stringified, URI encoded (to handle special characters), and then Base64 encoded to be passed in the URL hash.
